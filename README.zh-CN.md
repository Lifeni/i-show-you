### 前提

- 前端部分: Node 14+, Yarn
- 后端部分: Go 1.15+, MongoDB

### 配置

#### 前端

进入项目文件夹，并安装依赖：

```shell script
cd webapp && yarn
```

启动开发服务器：

```shell script
yarn start
```

#### 文档

进入项目文件夹，并安装依赖：

```shell script
cd docs && yarn
```

启动开发服务器：

```shell script
yarn docs:dev
```
